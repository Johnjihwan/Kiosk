# Kiosk
**식당의 키오스크를 JAVA(Swing)을 사용하여 개발**

>키오스크 기능 설명입니다.
```
1. 상품을 select할 수 있습니다.
2. 상품의 개수를 활용하여 가격을 계산합니다.
3. 상품 명세서를 csv 파일로 변환하여 제공합니다.
```
